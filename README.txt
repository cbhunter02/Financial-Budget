FINANCIAL RECOVERY DASHBOARD

How to use:
1. Unzip this folder.
2. Open index.html in a browser.
3. For best iPhone experience, upload the folder to Netlify, GitHub Pages, or any simple static host.
4. Once it opens in Safari, tap Share > Add to Home Screen.

Important:
- Your edits save in your browser using localStorage.
- If you open this from iOS Files, iOS may block JavaScript. Hosting it online is the most reliable method.
- The app is fully static. No account, no server, no private data leaves your device/browser.
